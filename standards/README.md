# Standards Reference Index

This directory contains agent-readable working summaries of standards relevant to Sifab AS operations. These are NOT copies of the standards themselves — they are structured references that help AI agents identify which clauses, requirements, and cross-references apply to Sifab AS scope of work.

## Directory Structure

| Folder | Contents |
|--------|----------|
| [norsok/](norsok/README.md) | NORSOK standards (Norwegian petroleum industry) |

## How Agents Use These Summaries

Each summary file follows a consistent template containing:

- **Standard identification** — number, title, edition
- **Scope** — what the standard covers
- **Key clauses for Sifab AS** — clauses relevant to flow metering, tank gauging, provers, skid fabrication, and offshore equipment
- **Agent mapping** — which Sifab agents reference this standard and why
- **Cross-references** — links to related standards

## Applicable Regulatory Framework

Sifab AS operates under Norwegian petroleum regulations enforced by:

- **Petroleumstilsynet (PSA)** — Petroleum Safety Authority Norway
- **Arbeidstilsynet** — Norwegian Labour Inspection Authority
- **DSB** — Directorate for Civil Protection
- **OD (formerly NPD)** — Norwegian Offshore Directorate

Sifab AS is certified to **ISO 9001**, **PED**, and **TR 2000**.
