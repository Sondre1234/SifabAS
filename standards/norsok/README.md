# NORSOK Standards — Agent Reference Index

NORSOK standards are developed by the Norwegian petroleum industry to ensure adequate safety, value-adding, and cost-effectiveness for petroleum industry developments and operations. They are managed by Standard Norge.

This directory contains agent-readable summaries of NORSOK standards relevant to Sifab AS scope of work: flow metering, tank gauging, volume provers, skid fabrication, and special items for offshore/onshore installations.
