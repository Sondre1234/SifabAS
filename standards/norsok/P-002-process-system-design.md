# NORSOK P-002 — Process System Design

| Field | Value |
|-------|-------|
| Standard | NORSOK P-002 |
| Title | Process System Design |
| Edition | Ed. 5, 2014 |

## Scope

NORSOK P-002 provides requirements for the detailed design of process systems on petroleum installations. It covers equipment sizing, drain systems, vent systems, sampling systems, chemical injection, and system-specific design requirements. The standard complements P-001 with detailed design rules for individual process systems.
