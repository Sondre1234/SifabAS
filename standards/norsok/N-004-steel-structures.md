## Scope

NORSOK N-004 provides rules for the design of steel structures for offshore petroleum installations. It covers ultimate limit state (ULS), serviceability limit state (SLS), fatigue limit state (FLS), and accidental limit state (ALS) design. The standard addresses structural members (beams, columns, tubulars), connections (bolted and welded), plates, and stiffened panels.
