# NORSOK Z-DP-002 — Coding System

## Scope
NORSOK Z-DP-002 establishes the coding system for tag numbering, equipment identification, document numbering, and area/system classification used across Norwegian Continental Shelf petroleum installations. It defines the structured alphanumeric coding conventions that ensure consistent identification of physical equipment, functional locations, P&ID tag numbers, and all engineering and operational documents throughout the facility lifecycle.

## Key Sections for Sifab AS
| Section | Topic | Relevance |
|---------|-------|-----------|
| §4 | Document numbering | Document number structure and discipline codes — Sifab uses this to assign document numbers to all project deliverables (e.g., drawings, datasheets, reports) |
| §5 | Tag numbering | Equipment and instrument tag number structure — Sifab must follow client tag conventions for instruments, valves, and metering equipment |
| §6 | Discipline codes | Discipline code definitions (IA, ME, ST, PR, PI, etc.) — used in Sifab document numbering convention |
| §7 | Document type codes | Document type codes (REP, DRW, CAL, DAS, PRO, etc.) — applied to all Sifab-issued documents |
| §8 | System and area codes | System numbering and area classification codes — metering, process, utility systems; location codes on facility |
| §9 | Equipment codes | Equipment type codes for flow meters, provers, tanks, pumps, valves, instruments — used in tag lists and MDR |
| §10 | Revision coding | Revision designation convention — numeric for issued-for-review, alpha for issued-for-construction |

## SifabAS Agent Usage
- **Primary**: Project Manager — applies Z-DP-002 coding to all project documents, tag lists, and document registers; enforces consistent numbering across the project
- **Secondary**: All discipline agents (Instrumentation, Process, Structural, Metering, HSE) — use discipline codes and document type codes when numbering their own deliverables; Quality Management Agent — document register and MDR compliance with Z-DP-002 structure; ISO Document Producer Agent — internal document numbering for QMS procedures and forms
