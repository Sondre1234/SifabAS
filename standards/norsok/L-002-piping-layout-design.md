# NORSOK L-002 — Piping System Layout, Design and Structural Analysis

| Field | Value |
|-------|-------|

| Title | Piping System Layout, Design and Structural Analysis |
| Edition | Ed. 4, 2019 |

## Scope

NORSOK L-002 covers requirements for piping layout, pipe support design, flexibility analysis, and structural analysis of piping systems on petroleum installations. It addresses routing principles, support types and spacing, thermal expansion, vibration, slug flow loads, and pipe stress analysis methodology.

## Key Clauses for Sifab AS

| Clause | Topic | Relevance to Sifab |
|--------|-------|---------------------|
| 5 | Layout principles | Routing and arrangement of piping on metering skids — accessibility, maintainability, flow direction |
| 6 | Pipe supports | Support type selection, spacing, and design for metering spools and prover piping — spring hangers, guides, anchors |
| 7 | Flexibility analysis | Thermal expansion analysis for piping on metering skids — prover pipe thermal growth is critical |
| 8 | Vibration | Flow-induced vibration assessment for metering runs — turbulence downstream of flow meters, valve noise |
| 9 | Slug and multiphase flow | Dynamic loads from slug flow on piping — relevant for wet gas metering systems |
| 10 | Nozzle loads | Equipment nozzle load verification — flow meter flange loads, prover connection loads, vessel nozzle loads |
| 11 | Pipe stress analysis | Stress analysis methodology — code compliance (ASME B31.3), load cases, allowable stresses |
| 12 | Ergonomic requirements | Access for maintenance, valve operation, instrument access — layout of metering skid walkways and platforms |
